Este projeto está dividido em 3 branches de acordo com a disciplina

- `Telas`: arquivos da **entrega 03/2.1**, de acordo com a imagem 1.
- `CRUD-ORM`: arquivos da _aplicação CLI_ abordada na **entrega 03/2.2**.
